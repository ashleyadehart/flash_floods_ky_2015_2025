# Have Flash Flooding Events Increased in Frequency Within the State of Kentucky from 2015-2025?

This project explores trends in flash flooding events in Kentucky from 2015–2025. The main question that was explored was whether flash flooding events had increased in frequency over the past ten years. This project demonstrates data cleaning, exploratory analysis, and visualization in Python.

## How to Use
1. Clone this repository.
2. Install the required Python packages:  
   `pip install -r requirements.txt`
3. Open `flash_floods_ky_2015_2025.ipynb` in Jupyter Notebook or JupyterLab.

## Output
The analysis shows patterns in flash flooding events over time, including yearly fluctuations in event frequency, counties with the highest number of events, and the most common reporting sources such as emergency services and call centers.

## Data Sources
- National Weather Service (NOAA Storm Events Database)
- Weather API
- Datasets used in this project can be found in the `data/` folder
